Analisis Redes Neuronales

Las Redes Neuronales han sido un gran avance en la magnitud de aprendizaje tanto de las maquinas como de nosotros las personas, la funcion principal de estas redes es imitar el proceso de aprendizaje y acabar desarollando una "cosa" que funcione, teniendo en cuenta que la idea original de estas Redes Neuronales es reproducir el funcionamineto del cerebro humano, el funcionamineto de estas Redes Neuronales consiste en imitar el funcionamineto de las redes neuronales de los organismos vivos, son un conjunto de neuronas conectadas entre si y que trabajan en conjunto, sin que haya una tarea concreta para cada una, Una Red Neuronal, termina siendo una Combinacion de parametros, con un fin dado, predecir un resultado, se podria entender como "entrenar" el ejercicio de modelar la red neuronal con el fin de encontrar dicha combinacion de parametros. Se entiende como Percepton, el elemento que tiene varias entradas con un cierto peso cada una, la forma de evaluar esto es con un condicial, por ejemplo, SI, se cumple tal condicion su valor retornado sera 1, een este caso Verdadero, De no ser asi, Retornara Falso, en este caso 0, lo mismo para con el Percepton, su funcionamineto se basa con pesos, y se evaluan los parametros, en caso de no dar el valor esperado, se ajustan minusciosamente los parametros, a esto es lo que se le llamaria como "entrenar" la red Neuronal, en Conclusion, las Redes Neuronales son una herramienta utilizada por el hombre para imitar el comportamiento de si mismo, su objetivo es concreto, imitar el procesamiento humano, predecir resultados y aprender de manera semi autonoma, bajo mi prespeectiva es semi autonoma, ya que aun para que llegar a un funcionamineto perfecto debe ser ajustada minuscionamente por mano humana.
